Proceso de Compra: 

1) seleccionar inicio  para ver catalogo completo o filtrar por mesas o productos desde el menu de navegacion.
2) una vez visualizado el producto de interes presioamos "Ver mas" para acceder al mismo.
3)En caso de querer comprarlo indicamos la cantidad de unidades deseadas y presionamos "Agregar al carrito" o volvemos al caralogo para seguir comprando.
4)En caso de haber avanzado en nuestra compra presionamos ir al carrito, luego finalizar compra en caso de que no quisieramos seguir comprando otros productos.
